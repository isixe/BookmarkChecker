# BookmarkChecker

BookmarkChecker is a tool for validating the availability of bookmark links. It can be used to check the availability of exported bookmarks, find broken links, and help you keep your bookmark list healthy and usable.

## Main Features

- Batch-check the validity of bookmark URLs (HTTP status codes, timeouts)
- Support importing from common bookmark formats and exporting to Excel, CSV, TXT, JSON, HTML, or TXT formats

## Quick Start

Install dependencies first:

```bash
pnpm install
```

Start the development environment:

```bash
pnpm run dev
```

The server will start at http://localhost:3000.
